# Sentinel - DNS Switcher

تمرکز برنامه روی انتخاب هوشمند بهترین DNS بر اساس پینگ، جیتر، پکت‌لاس و نیاز کاربر (استفاده عادی یا بازی آنلاین) است.

---

## ویژگی‌ها

- تشخیص خودکار اجرای برنامه با سطح دسترسی Administrator برای اعمال تنظیمات شبکه.  
- رابط کاربری تیره (Dark) و مدرن با استفاده از `customtkinter` و فونت فارسی اختصاصی.  
- دسته‌بندی DNSها به سه گروه: ایرانی (local)، جهانی (global) و سفارشی (custom).  
- افزودن، ویرایش و حذف DNSهای سفارشی توسط کاربر با اعتبارسنجی IPها.  
- تست سریع پینگ یک DNS یا همه DNSها به صورت نرمال و پیشرفته (پینگ، جیتر، پکت‌لاس و امتیاز).  
- رتبه‌بندی DNSها بر اساس نتایج تست کامل و نمایش آن در یک پنجره مجزا.  
- نمایش بهترین DNS پیشنهادی برای هر بازی در تب «بازی‌ها» بر اساس نتایج تست.  
- نمایش DNS فعلی سیستم و وضعیت کارت شبکه فعال.  
- ابزارهای سریع شبکه: ریست DNS (Flush) و ریست تنظیمات شبکه با هشدار و پیام نتیجه.  
- ذخیره و بارگذاری لیست DNSها و بازی‌ها از طریق فایل‌های JSON در پوشه برنامه.
