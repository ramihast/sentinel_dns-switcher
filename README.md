# 🎮 DNS بهینه‌ساز - README

## 📋 ویژگی‌های اصلی

| ویژگی | توضیح |
|--------|--------|
| 🎯 **پینگ تست زنده** | تست پینگ با نمایش نتایج |
| ⚙️ **مدیریت کامل DNS** | اضافه/ویرایش/حذف |
| 🎮 **بهینه‌سازی بازی** |  بهینه برای بازی های محبوب DNS|
| 🌐 **پشتیبانی IPv4/IPv6** | انتخاب پروتکل دلخواه |
| 🖥️ **انتخاب اینترفیس** | پشتیبانی از تمام کارت‌های شبکه |
| 🚀 **اعمال سریع** | ست کردن DNS با یک کلیک |


## 📁 ساختار فایل‌ها

```
📁 assets/
  ├── Dana-Regular.ttf    # فونت فارسی
  └── icon.ico           # آیکون برنامه

📄 dns_list.json         # لیست DNS ها
📄 games_list.json       # تنظیمات بازی‌ها
📄 main.py              # فایل اصلی
```


**⚠️ نکته مهم**: برنامه **حتماً** با دسترسی Administrator اجرا شود.

## 🎯 نحوه استفاده سریع

```
1️⃣ اینترفیس Wi-Fi را انتخاب کنید
2️⃣ IPv4 را انتخاب کنید  
3️⃣ 📡 پینگ همه → بهترین را ببینید
4️⃣ روی [ست] کلیک کنید
5️⃣ ✅ تمام!
```

## ⚠️ نکات مهم

- بعد از تغییر DNS، مرورگر را Refresh کنید
- برای ریست شبکه، سیستم را Restart کنید
